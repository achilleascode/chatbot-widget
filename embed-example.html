NEO Chat Widget - Einbettung

Diese Seite zeigt, wie das NEO Chat Widget in eine Website eingebettet wird.

Option 1: Einfaches Script-Tag (Empfohlen)

Fügen Sie diesen Code vor dem schließenden </body> Tag ein:

<script src="https://chatbot-widget-1.vercel.app/embed.js"></script>

Option 2: Mit eigener Widget-URL

Falls Sie das Widget von einer anderen URL laden möchten:

<script> window.NEO_CHAT_URL = 'https://ihre-domain.com/widget/'; </script> <script src="https://chatbot-widget-1.vercel.app/embed.js"></script>

Option 3: Inline Code

Kopieren Sie diesen Code direkt in Ihre HTML-Seite:

<script> (function() { var WIDGET_URL = 'https://chatbot-widget-1.vercel.app/'; var iframe = document.createElement('iframe'); iframe.id = 'neo-chat-widget'; iframe.src = WIDGET_URL; iframe.style.cssText = 'position:fixed;bottom:0;right:0;width:100px;height:100px;border:none;background:transparent;z-index:2147483647;transition:width 0.3s ease,height 0.3s ease'; iframe.setAttribute('allowtransparency', 'true'); window.addEventListener('message', function(e) { if (e.data && e.data.type === 'neo-chat-resize') { iframe.style.width = e.data.width + 'px'; iframe.style.height = e.data.height + 'px'; } }); document.body.appendChild(iframe); })(); </script>
